# 🎯 Decision Justifier (Smart Decision Maker)

## 📌 Overview
Decision Justifier is a Python-based GUI application that helps users make decisions when they are confused between multiple options.  
It takes user input, processes the choices, and selects one option while providing a logical justification.

---

## 🚀 Features
- 🧠 Smart decision-making using predefined logic
- 🎲 Random selection of choices
- 🧹 Input cleaning using `strip()`
- ⚠️ Input validation (handles empty input)
- 🖥️ Simple and user-friendly GUI using Tkinter

---

## 🛠️ Technology Stack
- **Programming Language:** Python 3.x
- **Library Used:** Tkinter (GUI)
- **Module Used:** random

---

## ⚙️ How It Works
1. User enters multiple choices separated by commas  
2. Input is processed and cleaned  
3. Program selects one option randomly  
4. A justification is generated using conditions  
5. Result is displayed on the GUI
